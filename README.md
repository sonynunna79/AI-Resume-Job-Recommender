# AI Resume Analyzer & Job Recommendation System 🤖

## Overview

This project uses Artificial Intelligence and Machine Learning to analyze resumes and recommend suitable job roles based on skills.

The system extracts resume content, compares skills with available job descriptions, and provides the best matching job recommendation.

## Features

✅ Upload Resume PDF  
✅ Extract Resume Text  
✅ AI Based Skill Matching  
✅ Job Recommendation  
✅ Simple User Interface using Streamlit


## Technologies Used

- Python
- Machine Learning
- Natural Language Processing (NLP)
- Streamlit
- Scikit-learn
- Pandas
